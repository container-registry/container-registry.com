About

A no frills Harbor based Container Management Solution for teams and organizations

Container Registry started out in 2016 as a Portus Powered Container Registry Service for teams and organizations. In September 2020 we switched the software stack to Harbor as the new foundation to better support cloud native and multi cloud workloads.

Our Background

Many solutions arise from their own and immediate needs. We were building a new microservice-based software stack for a new project and faced for the second time in a row the difficulty to find a decent Docker Registry suitable for teams and organizations. Docker Hub was the most used option, but lacked features we have been looking for. At the same time, we stumbled upon Portus, a new open-source Container Registry designed specifically for teams and organizations. This discovery sparked our interest to migrate an op-prem application to a SaaS, allowing us to learn more about selling and promoting a SaaS instead of only building and operating SaaS. 8gears helps software vendors build SaaS faster by providing state-of-the-art SaaS infrastructure with solutions for subscription management, authentication, authorization, multi-tenancy, Cloud rollout and operation.

The Future

Although Container Registries have now become largely a commodity, there are still businesses segments that won’t see a solution from Cloud Providers anytime soon:

  • Container Registry for teams and organizations, where access and Container Management is not entangled with the Cloud Provider’s IAM.
  • Businesses who run container workloads outside the big three Cloud Hyperscaler or on edge.
  • Businesses that pursuing Hybrid Cloud strategy
  • Independent Software Vendors who are distributing their software through Container Images instead of binaries or ZIP files.

We hope that these businesses will consider our services to be valuable.
